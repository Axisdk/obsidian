# 🧪 Тестирование контента на stage  
  
## 🔗 Доступ к стенду  
  
- Сайт: https://stage.investlink.kz  
- CMS (WordPress): https://stage.investlink.kz/login  
  
## 🔐 Авторизация  
  
1. Перейдите по ссылке CMS  
2. Введите свои учетные данные  
3. Выполните вход  
  
---  
  
## 📝 Создание записи  
  
1. В левом меню выберите:  
**Записи → Добавить запись**  
2. Создайте новую запись  
3. Выберите нужную рубрику (категорию)  
  
---
